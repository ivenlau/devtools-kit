## 🔍 快速查找

### 我想...

**开始开发**
1. 阅读 [README](../README.md) 了解项目
2. 查看 [01-产品概述与技术架构.md](./01-产品概述与技术架构.md) 了解技术栈
3. 参考 [02-P0工具详细规格.md](./02-P0工具详细规格.md) 开始实现工具

**添加新工具**
1. 在 [PRD.md](../PRD.md) 的功能模块中查找工具分类
2. 参考 [02-P0工具详细规格.md](./02-P0工具详细规格.md) 中的工具实现模式
3. 使用 [04-工具函数库规格.md](./04-工具函数库规格.md) 中的工具函数

**创建新组件**
1. 参考 [03-公共组件库规格.md](./03-公共组件库规格.md) 中的组件规范
2. 遵循 [01-产品概述与技术架构.md](./01-产品概述与技术架构.md) → 三、界面设计规范

**编写测试**
1. 参考 [05-测试策略与开发流程.md](./05-测试策略与开发流程.md) 的测试策略
2. 查看单元测试、集成测试、E2E测试的示例代码

**部署上线**
1. 查看 [README.md](../README.md) → 部署章节
2. 参考 [01-产品概述与技术架构.md](./01-产品概述与技术架构.md) → 5.1 构建配置
3. 遵循 [05-测试策略与开发流程.md](./05-测试策略与开发流程.md) → 九、发布流程
